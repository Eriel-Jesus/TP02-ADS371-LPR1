# Trabalho Prático 02 - Java (ADS371)

Este repositório contém as resoluções do **TP2** da disciplina de **Linguagem de Programação 1**. Abaixo estão os enunciados e as demonstrações visuais de cada exercício.


## Exercícios e Demonstrações

### Exercício 01
Entrar com dois valores via teclado, onde o segundo valor deverá ser maior que o primeiro. Caso contrário solicitar novamente apenas o segundo valor.
![Demonstração Exercício 01](./TP2%20-%20Eriel_Brandon%20-%20ADS371%20-%20LPR1%20-%20Java/ex01.gif)

---

### Exercício 02
Entrar via teclado com dez valores positivos. Consistir a digitação e enviar mensagem de erro, se necessário. Após a digitação, exibir: o maior valor, a soma e a média aritmética.
![Demonstração Exercício 02](./TP2%20-%20Eriel_Brandon%20-%20ADS371%20-%20LPR1%20-%20Java/ex02.gif)

---

### Exercício 03
Entrar com “N” valores (0 < N < 20). Exibir maior, menor, soma, média e porcentagem de positivos/negativos. Possui opção de nova execução (S/N).
![Demonstração Exercício 03](./TP2%20-%20Eriel_Brandon%20-%20ADS371%20-%20LPR1%20-%20Java/ex03.gif)

---
